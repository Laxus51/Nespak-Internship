
# SQL Practice

This folder contains SQL practice completed during my WebGIS Internship.

The goal is to strengthen practical SQL skills while learning PostgreSQL and preparing for backend development with FastAPI.

The SQL scripts are organized by topic and are executed against a University Management System practice database.

## Files

| File                    | Description                                             |
| ----------------------- | ------------------------------------------------------- |
| schema.sql              | Database schema creation                                |
| sample_data.sql         | Sample data for testing queries                         |
| joins.sql               | INNER, LEFT, RIGHT and FULL joins                       |
| aggregate_functions.sql | COUNT, SUM, AVG, MIN, MAX, GROUP BY, HAVING             |
| subqueries.sql          | Nested queries                                          |
| ctes.sql                | Common Table Expressions                                |
| window_functions.sql    | RANK, DENSE_RANK, ROW_NUMBER and other window functions |
| views.sql               | Creating and querying SQL Views                         |
| indexes.sql             | Index creation and indexing strategies                  |
| optimization.sql        | EXPLAIN and EXPLAIN ANALYZE examples                    |
| postgres_functions.sql  | Common PostgreSQL built-in functions                    |

## Topics Practiced

- Database Schema Design
- Constraints
- Primary & Foreign Keys
- SQL Queries
- Joins
- Aggregate Functions
- Grouping
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Views
- Indexes
- Query Optimization
- PostgreSQL Functions

## Practice Database

All SQL examples use a University Management System consisting of the following entities:

- Departments
- Teachers
- Students
- Courses
- Enrollments

This database provides realistic relationships for practicing SQL queries and backend database operations.
